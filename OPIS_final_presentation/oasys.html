Oasys Feature

Overview

Internal System's web application Oasys functions as an administration hub for several systems of OPIS. In Oasys's Ougoing Email Log section, there are several filters such as date, recipient email, customer ID, etc. However, there has never included a filter for product ID. I added this feature so that the application includes a field for product ID and results are filtered by it if one is provided.
