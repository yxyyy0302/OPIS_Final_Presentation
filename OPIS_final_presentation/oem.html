OEM Update

Description

The Opis Entitlements management (OEM) system is a service developed to manage licensing and user-level roles and permissions in various OPIS services. Similar to the Mend projects, OEM's Node.js version is too old. I updated Node.js's version so that OEM authorizer lambda now targets the latest Node.js runtime supported by AWS (nodejs16.x -> nodejs22.x).
